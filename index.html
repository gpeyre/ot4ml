Optimal Transport
for Machine Learners

This website is the public companion to OT4ML. It is not only a book page: it gathers the manuscript, an experimental interactive MyST version, the executable notebooks that regenerate the book figures, teaching notebooks, course slides, and curated references/software for learning and using optimal transport in machine learning.
