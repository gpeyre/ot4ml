# Removed Figure Notebooks

This directory contains figure-generation notebooks that are no longer used by
the current OT4ML manuscript. They were moved out of `notebooks-figures/`
because their figure names are not referenced by live
`\includegraphics{figures/...}` commands in `latex/`.

There are currently 18 archived notebooks here. Their matching generated PDF
panels are kept in `../../latex/figures/removed/`, and archived thumbnails are
kept in `thumbnails/`. These notebooks are retained for provenance and possible
future reuse, but they are intentionally omitted from the paper figure gallery.
